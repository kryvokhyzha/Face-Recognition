I. Встановлення інтерпретатора Python-3.5.4:
	a) Перейдіть до кореневої папки програми (../face-recognition-flask-site/)
	b) Відкрийте папку setup
	c) Для тихого встановлення запустіть Install Python 3.5.4.bat, двічі натиснувши лівою кнопкою миші (рекомендовано встановлювати таким чином)
	d) Дочекайтесь завершення встановлення та натисність Enter

	Примітка: приблизний час встановлення 5хв


II. Встановлення всіх необхідних пакетів (необхідне підключення до інтернету):
	a) Перейдіть до кореневої папки програми (../face-recognition-flask-site/)
	b) Відкрийте папку setup
	c) Для тихого встановлення всіх пакетів та оновлення версії pip запустіть з
	   правами адміністратора файл Install packages and upgrade pip.bat, натиснувши на файл
	   правою кнопкою миші та обравши пункт "Запустити з правами адміністратора" (рекомендовано встановлювати таким чином)
	d) В разі виникнення помилки, проведіть ці операції окремо, спочкатку запустивши
	   з правами адміністратора файл Upgrade pip.bat, а тоді Install packages.bat
	e) Дочекайтесь завершення встановлення та натисність Enter

	Примітка: приблизний час встановлення 10-15хв.


III. Запуск програми:
	a) Перейдіть до кореневої папки програми (../face-recognition-flask-site/)
	b) Запустіть в кореневій папці проекту файл START.bat, двічі натиснувши на нього лівою клавішою мишки
	с) Відкриється консоль та почне розгортатись сервер
	d) Сервер буде запущеним після того, як в консолі з'явиться наступне повідомлення:
 		* Serving Flask app "app" (lazy loading)
 		* Environment: production
   		WARNING: Do not use the development server in a production environment.
   		Use a production WSGI server instead.
 		* Debug mode: on
 		* Restarting with stat
 		* Debugger is active!
 		* Debugger PIN: 295-741-000
		* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
	e) Через декілька секунд буде запущено браузер з початковою сторінкою програми.
	   Якщо це не відбудеться, то перейти до пунктів f, а тоді до пункту g.
	   Якщо головна сторінка все ж відкриється, то перейти до пункту g.
	f) Відкрийте браузер та перейдіть за посиланням, вказаним в останньому виведеному повідомленні.
	   Зазвичай це посилання наступне: http://127.0.0.1:5000/
	g) Відкриється головна сторінка з інформацією про програму та інструкцією по її використанню
	h) Для завершення роботи сервера сфокусуйтесь на вікні консолі, а тоді натисніть
	   комбінацію клавіш CTRL+C, далі з'явиться наступне повідомлення:
		Terminate batch job (Y/N)?
	   Вводимо в консоль Y, а далі просто натискаємо ENTER
		
	Примітка: в правій частині екрану головної сторінки знаходиться інструкція по користуванню сайтом,
		  а в лівій частині зхаходиться інструкція по тому як додавати зображення в базу та
		  інструкція по встановленню ПЗ.
