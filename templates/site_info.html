Встановлення інтерпретатора Python-3.5.4

  1. Перейдіть до кореневої папки програми (../face-recognition-flask-site/)

  2. Відкрийте папку setup

  3. Для тихого встановлення запустіть Install Python 3.5.4.bat

  4. Для звичайного встановлення оберіть python-3.5.4-amd64.exe (не забудьте поставити галочку біля пункуту PATH)


Встановлення всіх необхідних пакетів (необхідне підключення до інтернету)

  1. Перейдіть до кореневої папки програми (../face-recognition-flask-site/)

  2. Відкрийте папку setup

  3. Для тихого встановлення всіх пакетів запустіть Install all package.bat


Також передбачений спосіб встановлення і інтерпретатора, і пакетів (необхідне підключення до інтернету)

  1. Перейдіть до кореневої папки програми (../face-recognition-flask-site/)

  2. Відкрийте папку setup

  3. Для тихого встановлення всіх пакетів запустіть Install all.bat


Також можливий варіант копіювання вже встановленого інтерпретатора (дуже мала вірогідність сумісності з системою)

  1. Перейдіть до кореневої папки програми (../face-recognition-flask-site/)

  2. Відкрийте папку setup

  3. Скопіюйте папку Python35

  4. Вставте дану папку за шляхом C:\Users\User_name\AppData\Local\Programs\Python\

  5. Скопіюйте всі файли, окрім папки setup, з кореневої папки програми та вставте за шляхом C:\Users\User_name\AppData\Local\Programs\Python\


Для запуску сайту запустіть файл START.bat
