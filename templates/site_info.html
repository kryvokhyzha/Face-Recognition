Встановлення інтерпретатора Python-3.5.4

  1. Перейдіть до кореневої папки програми (../face-recognition-flask-site/)

  2. Відкрийте папку setup

  3. Для тихого встановлення запустіть Install Python 3.5.4.bat (рекомендовано встановлювати таким чином)

  4. Для звичайного встановлення оберіть python-3.5.4-amd64.exe (не забудьте поставити галочку біля пункуту PATH)


Встановлення всіх необхідних пакетів (необхідне підключення до інтернету)

  1. Перейдіть до кореневої папки програми (../face-recognition-flask-site/)

  2. Відкрийте папку setup

  3. Для тихого встановлення всіх пакетів та оновлення версії pip запустіть в режимі адміністратора файл Install packages and upgrade pip.bat (рекомендовано встановлювати таким чином)

  4. В разі виникнення помилки, проведіть ці операції окремо, спочкатку запустивши в реживі адміністратора файл Upgrade pip.bat, а тоді Install packages.bat


Для запуску сайту запустіть в кореневій папці проекту файл START.bat
